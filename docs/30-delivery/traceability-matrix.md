# Traceability Matrix

| Requirement | Phase | Verification |
| --- | --- | --- |
| Create intake | Sprint 1 / Phase 4 | Create intake test, empty state test, mobile form test. |
| Convert intake to service request | Sprint 1 / Phase 5 | Conversion test, duplicate customer by phone test, preserved vehicle/location test. |
| Create service request directly | Phase 5 | Direct create test and validation test. |
| Show lifecycle timeline | Phase 5 | Status transition audit log test and related records display test. |
| Manage service catalog | Sprint 2 / Phase 6 | Create, edit, deactivate, and default active selection tests. |
| Create estimate from service request | Sprint 2 / Phase 7 | Inherited data, line items, totals, and approval threshold tests. |
| Capture approval | Sprint 2 / Phase 8 | Approval record creation, estimate status update, timeline display. |
| Create work order | Sprint 3 / Phase 9 | Work order from approved estimate and status tests. |
| Use mobile field screen | Sprint 3 / Phase 9 | Mobile viewport test and dispatch timestamp test. |
| Complete service report | Sprint 3 / Phase 10 | Notes, VIN, no-vehicle-serviced flag, attachments, completion status. |
| Generate invoice | Sprint 4 / Phase 11 | Line copy, totals, VIN rule, approval threshold rule, PDF generation. |
| Record payment | Sprint 4 / Phase 12 | Full payment, partial payment, balance update, overpayment handling. |
| Generate receipt | Sprint 4 / Phase 12 | Receipt record and receipt document test. |
| Post accounting entries | Sprint 4 / Phase 14 | Balanced ledger tests for invoice and payment. |
| Review vendor document | Sprint 5 / Phase 13 | Upload, categorize, approve, post, and non-business review tests. |
| Generate proof packet | Sprint 5 / Phase 15 | Complete packet and missing-item packet tests. |
| View reports | Sprint 5 / Phase 16 | Empty data, sample data, filter, and total reconciliation tests. |
